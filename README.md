# Bootstrap Rails 5 API

Bootstrap Ruby on Rails 5 API (only)

- Ruby 2.2.2+
- Rails 5
- Postgresql
- Rack CORS
- Rack Attack
- Active Model Serializers
- RSpec
- Factory Girls
- Puma

### Prerequisities

```
Ruby 2.2.2+
Rails 5
```

### Installing

```
bundle install
rake db:setup
```

And run server

```
rails s
```

## Running the tests

```
curl -H "Authorization: Token token=<TOKEN>" http://localhost:3000/v1/users
```
